# Encryption-Decryption-DSKC-26

A multimedia encryption suite built during DSKC Summer Internship Programme 2026
at Miranda House, Department of Physics.

## What it does
- Text encryption using Substitution Cipher
- Image encryption using XOR
- File encryption using Fernet (AES)

## How to run
Clone this repository, then:
pip install -r requirements.txt

## Programs
- encrypt.py — Substitution cipher for text
- image_encrypt.py — XOR encryption for images
- fernet_encrypt.py — Fernet/AES encryption for any file

## Built with
Python, Pillow, NumPy, Matplotlib, Cryptography
